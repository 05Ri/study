**과제 참여 방법**

1. 저장소를 fork 한다.
2. fork된 자신의 레파지토리를 clone한다.
3. 생성된 폴더 내에 Github ID로 폴더를 생성한다.
4. 해당 주차 폴더 내에 코드를 작성한다.
5. 원본 저장소로 PR를 보낸다.
